# Bayesian Portfolio

This repository documents my journey in Bayesian statistics at Biostatistics & Data Science Innovation Lab ([BDSi](https://sites.google.com/view/biostat-ds-innovation/home)), NCC-GCSP, Republic of Korea.

This repository covers topics in Bayesian data analysis and modeling using Python, R and Stan

For Python environment, you can install all necessary packages with `environment.yml` file using [conda](https://docs.conda.io/en/latest/)

```bash
conda env create -f environment.yml
```
